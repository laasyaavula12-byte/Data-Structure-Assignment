## Assignment Description ##

This repository contains C programs developed as part of the *Introduction to Data Structure* course. The assignment focuses on implementing core data structure concepts such as stacks and queues and applying them to solve practical problems.

---

## Programs Included ##

### 1. Reverse String using Stack

* Demonstrates stack (LIFO) principle
* Reverses a string using push and pop operations

### 2. Parenthesis Matching

* Checks whether an expression is balanced
* Uses stack to validate opening and closing brackets

### 3. Next Greater Element

* Finds the next greater element for each element in an array
* Uses traversal-based comparison

### 4. Queue Operations (Linear Queue)

* Implements basic queue operations:

  * Enqueue
  * Dequeue
  * Display
* Uses array representation (FIFO principle)

### 5. Circular Queue

* Implements circular queue using arrays
* Uses modulo operation for efficient space utilization
* Supports enqueue, dequeue, and display operations
